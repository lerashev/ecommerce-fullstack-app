# Full stack E-commerce store

Online store application built from scratch using React, Node and PostgreSQL

## Technologies used

### Front end:

-   React.js
-   React Router
-   MobX state management
-   Bootstrap
-   Axios

### Back end:

-   Node.js
-   Express
-   PostgreSQL
-   Sequelize
-   JWT authentication

## Functionality

-   Collection Page
-   Product page
-   Filter facets
-   Administration page
    -   Product management
    -   Category management
